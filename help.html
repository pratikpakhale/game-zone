Game Help
     MasterMind Game is a logical and strategy game helpful for improving calc's speed. It includes for cracking/finding out a hidden code which is later disclosed when game lost. Read the instructions carefully.


1] Only 4 Chances are given to crack the code. The code changes every time you refresh the page or when game's lost.

2] Once you type the number in the textbox it becomes lock until and unless you finsh typing all textboxes and submit your code.

3] When the color of the typed number changes to Red it means the number you have typed is not among the hidden code.

4] When the color of the typed number changes to Orange it means the number you have typed is among the hidden code but not exatly at the place where it's needed.

5] When the color of the typed number changes to Green it means the number you have typed is among the hidden code and is exatly at the place where it's needed.

6] The previous values get saved automatically at the "Previous Values" labled textboxes.

7] The Number can reapeat twice thrice or more as they are randomly placed.
